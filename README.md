# PUBG感度調整ツール iPad HTML5版

GitHub Pagesでそのまま公開できる単体HTML版です。

## 公開方法

1. GitHubで新規リポジトリを作成  
   例：`pubg-sensitivity-tool`

2. この中の `index.html` をアップロード

3. GitHubのリポジトリ画面で  
   `Settings` → `Pages`

4. `Build and deployment` の Source を  
   `Deploy from a branch` にする

5. Branch を  
   `main` / `/root` にして Save

6. 数分後にURLが出ます  
   例：`https://ユーザー名.github.io/pubg-sensitivity-tool/`

## iPadで使う

SafariでURLを開く → 共有 → ホーム画面に追加

## 注意

このツールは非公式の感度計算ツールです。公式ロゴ・公式画像は使用していません。
